# Employee Turnover Classification

A Machine Learning project that predicts whether an employee is likely to leave the company based on workplace and performance-related factors.

## Features

* Data Preprocessing
* Logistic Regression
* StandardScaler
* Scikit-Learn Pipeline
* 5-Fold Cross Validation
* Confusion Matrix
* Accuracy, Precision, Recall, and F1 Score Evaluation

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Jupyter Notebook

## Results

* Accuracy: 86.3%
* Precision: 86.7%
* Recall: 83.2%
* F1 Score: 84.9%
